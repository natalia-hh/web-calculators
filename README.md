# Web Calculators

A React + TypeScript collection of small trading calculators.

## Calculators

- Position Average Calculator: estimates the effect of adding to an existing same-direction position.
- Position Risk / Margin Calculator: placeholder folder for the next calculator.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Each calculator should keep domain logic inside its own folder under `src/calculators/`, while shared UI components live in `src/components/`.
